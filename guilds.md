# Life RPG Creative Guilds System

## Overview
Inspired by modular job systems (e.g., FF5 style base classes with ability inheritance and multiclassing). PrintForge is the Builder Guild anchor.

Guilds represent timeless human creative archetypes. Kids multiclass for hybrid mastery.

## Guild Matrix & Synergies

| Guild | Core Output | Key Skills | Synergies with Builder |
|-------|-------------|------------|------------------------|
| Builder (PrintForge) | Physical creations | Mechanics, iteration, verification | Base for all - props/tools |
| Storyweaver | Narratives | Worldbuilding, rhetoric | Documented inventions, lore for builds |
| Visual Alchemist | Images/Design | Visualization, aesthetics | Skins, concept art for prints |
| ... (expand) | ... | ... | ... |

## Progression Path
Novice → Adept → Master per guild + shared abilities pool.

## Decision: Central spec
Keep one guilds.md + subfolders for deep dives initially for speed. Later per-guild repos or folders.
